# 字体替换脚本问题说明
由于部分网页的 Shadow DOM 和 Content Security Policy（CSP）限制，如果想要字体生效就必须**在本地同步安装对应语种与字重的字体**，否则会出现显示异常，如字体替换不生效、排版错位等现象。

若进行如上操作后问题仍未解决，可尝试**在浏览器的设置中将``默认字体``设为想要替换的字体**再试。

# 安装字体说明
尽量安装完整的语种和字重，效果最好。

如果不在乎各语种间的写法以及显示差异，也不在乎粗细的话只安装 ``Regular`` 常规体即可。
