# 字体替换脚本问题说明
由于部分网页的Shadow DOM和Content Security Policy（CSP）限制，如果想要字体生效就必须**在本地同步安装对应语种与字重的字体**，否则会出现显示异常，如字体错位、排版错位、一部分字体替换成功一部分不成功等现象。

若进行如上操作后问题仍未解决，可尝试**在浏览器的设置中将``默认字体``设为想要替换的字体**以及**搭配[拓展](https://microsoftedge.microsoft.com/addons/detail/web-page-font-replacement/lkiifecdjjijjdceondieabmlndffgba)进行字体替换**（参见下文）。

---

# 拓展使用说明
**使用方法：**
1. 安装好插件后点击浏览器右上角的插件图标，在菜单中选择 **``扩展选项``**；
2. 打开 **``总开关``** 让插件生效；
3. 根据个人喜好配置需要被替换的字体。

**注意：**
1. 此插件不适用于``icon``图标字体替换；
2. <ins>要替换的字体必须在系统里面已经存在。</ins>
